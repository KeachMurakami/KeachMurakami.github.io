はまったのでメモ。

複数パラメータを引数にとる自作関数で感度分析するとき、出力のデータフレームには入力したパラメータがあってほしい。
引数が多くなると`result %>% mutate(input1, input2, ....) %>% return` が長くなるし、修正に不便なので、引数を一括で足したい。


```r
library(tidyverse)
library(rlang)

f1 <- 
  function(a = 3, b = 2, c){
    args <- as.list(environment())
    
    tibble(result = a + b + c) %>%
      mutate(rlang::UQS(args))
  }

f2 <-
  function(a = 3, b = 2, c){
    args <- as.list(match.call())[-1]
    
    tibble(result = a + b + c) %>%
      mutate(rlang::UQS(args))
  }
```
